# Design Judgment Output Contracts

Use these report shapes when the default contract in `SKILL.md` is too large or too small.

## Compact quick read

```markdown
## 判断
- 结论：...
- 置信度：high / medium / low
- 一句话原因：...

## 最关键的问题
1. [P1 high] ...
   - Evidence: ...
   - Impact: ...
   - Fix: ...
   - Confidence: ...

## 下一步
- 先改/先验证：...
- 我没有验证：...
```

- 判断对象：...
- 最主要风险：...

## 范围和证据

## Saved report validation

When the report is saved to Markdown, optionally run:

```bash
python3 skills/dbx-design-judgment/scripts/validate-design-report.py path/to/report.md
```

This checker only verifies report shape and evidence markers. It does not prove the design judgment is correct.
