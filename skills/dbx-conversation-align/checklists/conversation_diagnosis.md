# Conversation Diagnosis Checklist

Use this compact checklist at runtime.

## 1. Task type

- [ ] Analyze a stuck conversation
- [ ] Rewrite a risky message
- [ ] Decide whether to correct
- [ ] Prepare difficult conversation
- [ ] Set or escalate boundary
- [ ] Work alignment / decision
- [ ] Relationship repair

## 2. Hard gates

- [ ] Any immediate safety risk, threat, violence, self-harm, coercive control?
- [ ] Is the user asking for manipulation or compliance tactics?
- [ ] Is the issue legal/medical/financial/HR/high-stakes professional?
- [ ] Is the other person repeatedly ignoring boundaries or using contempt?
- [ ] Are we mind-reading from weak evidence?

If yes, adjust strategy before giving scripts.

## 3. Conversation function

- [ ] Venting
- [ ] Comfort-seeking
- [ ] Validation-seeking
- [ ] Analysis-seeking
- [ ] Advice-seeking
- [ ] Decision-making
- [ ] Conflict/defense
- [ ] Boundary issue

Probe if uncertain:

> 你现在更想让我听你说，还是一起分析/想办法？

## 4. Emotional temperature

- [ ] 0 calm
- [ ] 1 mild emotion
- [ ] 2 clear emotion
- [ ] 3 defensive/escalating
- [ ] 4 crisis/threat

## 5. Stuck layer

- [ ] Emotion
- [ ] Relationship
- [ ] Goal
- [ ] Facts
- [ ] Concepts
- [ ] Logic
- [ ] Values
- [ ] Interests
- [ ] Identity
- [ ] Action

## 6. Correction decision

- [ ] Does it affect safety, major decision, harm, or trust?
- [ ] Did the other person invite analysis?
- [ ] Is emotion low enough?
- [ ] Is the motive help, not discomfort with imprecision?

Decision:

- [ ] Do not correct now
- [ ] Light correction
- [ ] Delay correction
- [ ] Must correct

## 7. Output must include

- [ ] concise diagnosis
- [ ] recommended mode
- [ ] directly usable wording
- [ ] what to avoid
- [ ] boundary/safety note when relevant
