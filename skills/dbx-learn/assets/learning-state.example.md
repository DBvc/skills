
## MISSION.md example

```md
# Learning Mission: React Compiler

## Why

I want to understand React Compiler well enough to judge adoption risks and design safe experiments for a production frontend codebase.

## Success criteria

- I can explain the compiler's mental model and constraints.
- I can identify code patterns that block or reduce compiler benefits.
- I can design a small experiment with success and rollback signals.

## Non-goals

- I am not trying to memorize every implementation detail.
- I am not committing to production adoption yet.
```
