# English Version

English mode is transcreation, not literal translation.

## Goal

Preserve:

- thesis;
- argument structure;
- emotional temperature;
- author stance;
- technical precision;
- reader effect.

Change when needed:

- idioms;
- sentence order;
- examples;
- transitions;
- title;
- paragraph rhythm.

## Common Chinese-to-English problems

Avoid:

```text
With the rapid development of AI...
```

Prefer:

```text
AI has made it easier to produce a complete-looking draft, but harder to hide the absence of judgment.
```

Avoid:

```text
This is not only a technical problem, but also an organizational problem.
```

Prefer:

```text
The technical symptom is component design. The real bottleneck is who gets to define the boundary of state.
```

Avoid:

```text
We need to embrace change rationally.
```

Prefer:

```text
The useful question is not whether to embrace AI, but which parts of the workflow can survive cheaper generation and which still require human judgment.
```

## English tone options

Infer from target surface.

### Technical essay

- clear;
- concrete;
- argument-driven;
- restrained;
- no startup-brochure energy.

### Personal essay

- intimate but not overdramatic;
- specific;
- quiet;
- no therapy-speak unless user uses it.

### Public post

- shorter paragraphs;
- stronger opening;
- less conceptual scaffolding;
- one memorable ending.

## Workflow

1. Identify the Chinese thesis.
2. Decide whether the English reader needs additional context.
3. Rebuild paragraphs in English logic.
4. Preserve examples when they travel well.
5. Replace untranslatable metaphor with equivalent function.
6. Check technical terms.
7. Remove Chinglish skeletons.

## Bilingual output

When asked for bilingual output, use:

```markdown
## 中文

...

## English

...
```

Do not interleave paragraph-by-paragraph unless requested.

## Fact policy

Do not add global context, Western examples, citations, or market claims just to make the English version sound natural.

If a Chinese claim was uncertain, the English version must remain uncertain.
